# 品牌避雷查詢

這是一個本機靜態查詢頁。頁面讀取同資料夾的 `brands.json`，提供搜尋、篩選與詳情顯示。

## 開啟方式

請用本機 HTTP 靜態伺服器開啟，不要直接雙擊 `index.html`。

PowerShell：

```powershell
.\start-server.ps1
```

開啟：

```text
http://localhost:4173/
```

若 4173 已被使用：

```powershell
.\start-server.ps1 -Port 4174
```

## 資料

`brands.json` 目前有 61 筆資料：

- 使用者指定的個人避買／辨識字樣。
- 公開來源可支持的中共政權關聯風險廠商。
- 依證據強度標示 `high`、`medium`、`low`，狀態則區分 `來源確認`、`觀察中`、`個人避買`。

收錄門檻不是「中國製造」，而是至少有國企／黨政治理／官方制裁或管制／人權供應鏈／政府資安警示等公開來源之一。低信心或只有調查性質的項目應放 `watchlist`，不要寫成法律定罪。

資料欄位原則：

- `aliases` 只放品牌名、公司名、明確系列名或常見通路名稱。
- 沒有品牌限定的普通商品名，例如水果品種、鳳梨酥、熱水壺、行動電源，不放 `aliases`。
- 統一編號等數字放 `identifiers`，顯示為「辨識編號」，不混在別名。
- 搜尋框只查品牌名、別名與辨識編號；分類與避買理由用下拉篩選。
- `categories` 使用大類，不放細項。現有分類固定為：食品／飲料／農水產、科技／通訊／消費電子、監控／AI／資安、電商／App／數位服務、家電／智慧家庭、汽車／交通／能源、服飾／玩具／生活零售、餐飲／生活服務。

若要重跑本輪親共／中共政權關聯風險資料合併：

```powershell
node .\scripts\merge-risk-records.mjs
```
